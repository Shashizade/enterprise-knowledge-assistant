# Enterprise Knowledge Assistant

AI-powered enterprise knowledge platform built using Azure AI Foundry, Azure OpenAI, Azure AI Search, FastAPI and Streamlit.

## Features

- Semantic Search
- Vector Search
- RAG
- Citations
- Multi-format Document Processing
- Enterprise Security

## Supported Formats

- PDF
- DOCX
- XLSX
- PPTX
- TXT
- XML
- XSLT
- WSDL
- OpenAPI
- Logic Apps
- APIM Policies
