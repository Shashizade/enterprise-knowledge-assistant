# Enterprise Knowledge Assistant (EKA)

## Project Overview

Enterprise Knowledge Assistant is an AI-powered platform that enables users to search and interact with enterprise knowledge using natural language.

The solution leverages Azure AI Foundry, Azure OpenAI, Azure AI Search, FastAPI, and Streamlit to provide contextual and citation-based responses from organizational documents.

---

## Business Problem

Enterprise knowledge is distributed across multiple document types:

- PDF Documents
- Word Documents
- Excel Files
- PowerPoint Files
- XML
- XSLT
- WSDL
- OpenAPI Specifications
- Logic App Definitions
- API Management Policies

Finding information is time consuming and requires subject matter expertise.

---

## Objectives

- Enable semantic search across enterprise documents.
- Provide AI-generated responses with citations.
- Support conversational querying.
- Reduce knowledge retrieval time.
- Improve onboarding and support efficiency.

---

## Key Features

- Document Upload
- Document Processing
- Metadata Extraction
- Chunking
- Vector Search
- RAG-based Question Answering
- Authentication
- Monitoring

---

## Success Criteria

- Upload and process documents successfully.
- Generate embeddings and vector index.
- Return grounded responses.
- Provide source citations.
- Support secure authentication.
- Deployable to Azure.
